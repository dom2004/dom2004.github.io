My Projects

Here you will see a range of different projects that I have worked on throughout the years, ranging from simple game projects to big coding projects and renders.

Black Eagle

A simple platformer game created as a part of my Games Design assignment during my A Levels. This was created using Gamemaker Studio 2. The game features a main character, a ninja traversing through a dystopian, futuristic cyberpunk city, ruled by an organization that sets fear into its citizens eyes with its high-tech robo police. The main character, nicknamed 'Black Eagle' by the citizens for his stealthiness, is on a mission to finally liberate the city from tyranny and give freedom to the people, as they so longed for. You can view the source code in GitHub here.

Snake Game

This is a simple snake game using Microsoft's WPF framework and .NET C#. This was a simple project that helped me learn WPF development and basic XAML. Additionally it helped me strengthen my C# knowledge, thus getting me more comfortable with the language. You can view the source code for the project here.

OpenGL Bus Render

A simple OpenGL render created as a part of my Graphics Programming module. It utilises Vector and Matrix transformation structs, a scenegraph, simple lighting and an ImGui debug menu. You can view more information about the project here.
